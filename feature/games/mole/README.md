# Мини-игра «Поймай крота» (`:feature:games:mole`)

Игра для выключения будильника (game id = **23** в `ALL_GAMES`):
классическая whack-a-mole на сетке нор.

## Правила

- На экране квадратная сетка «нор» (🕳️).
- Крот (🐹) появляется в случайной норе на ограниченное время, затем
  прячется и через паузу (450 мс) выскакивает в другой норе — **никогда
  не в той же самой**, где был в прошлый раз (даже если успел уйти сам).
- **Тап по кроту** — попадание: +1 к прогрессу, крот сразу прячется.
- **Тап по пустой норе** (в том числе когда крота вообще нет на поле) —
  промах: −10 очков. Промахи не сбрасывают прогресс.
- Если крот **ушёл сам** (истекло время показа) — штрафа нет.
- Победа — набрать нужное число попаданий; после победы игра переходит
  к экрану результата.

На экране отображаются таймер, счётчик «Поймано: k/N» и число промахов.

## Сложности

| Уровень | Сетка | Крот виден | Нужно попаданий |
|--------:|:-----:|:----------:|:---------------:|
| 1       | 3×3   | 1200 мс    | 5               |
| 2       | 3×3   | 900 мс     | 7               |
| 3       | 4×4   | 650 мс     | 10              |

Пауза между появлениями — 450 мс, первая — через 600 мс после открытия
экрана (константы во фрагменте). Значения сложности вне 1..3 приводятся
к ближайшему уровню.

## Очки

Как в остальных играх (образец — калькулятор):

- каждый промах: **−10** очков (`MoleGame.MISS_PENALTY`);
- при победе: **+ (600 − прошедшие_секунды) × сложность**
  (чистая функция `finishBonus`; при игре дольше 10 минут бонус
  отрицательный).

Итог = сумма штрафов + финальный бонус.

## Архитектура

| Файл | Роль |
|------|------|
| `MoleLogic.kt` | **Чистый Kotlin, без Android.** `MoleDifficulty` (параметры уровня, `forLevel`), `MoleGame` (где крот, `showMole`/`hideMole`/`tap`, счёт попаданий/промахов, `isWon`; случайность — через параметр `kotlin.random.Random`, следующая нора равномерно выбирается из всех, кроме предыдущей), `MoleTapResult`, `finishBonus`. |
| `MoleGameViewModel.kt` | `AndroidViewModel` по образцу `CalcGameViewModel`: таймер («м.сс»), счёт очков, перезапуск будильника (`AlarmDbRepository`/`AlarmCreateRepository`), хранит `MoleGame` — прогресс переживает поворот экрана (`ensureGameCreated` не пересоздаёт игру). |
| `MoleGameFragment.kt` | UI и тайминги: строит сетку (`android.widget.GridLayout` программно, ячейки — `TextView` с эмодзи), цикл появления крота на `Handler.postDelayed` (запускается в `onResume`, останавливается в `onPause`), обработка тапов, контракт экрана игры. |
| `res/layout/fragment_mole_game.xml` | `ConstraintLayout`: таймер, счётчики, `GridLayout` (квадрат, `1:1`), кнопка выключения музыки. Цвета — через `?attr/...`. |

## Контракт экрана (см. `.claude/rules/games.md`)

- Аргументы: `alarm id` (Long), `start time` (Long, 0 = проба),
  `test` (Boolean), `difficulty` (Int 1..3), `music path` (String).
- `onResume`: если не `test` — снимается уведомление будильника;
  (пере)запускается цикл появления крота.
- `onPause`: цикл останавливается (колбэки `Handler` снимаются), крот
  прячется; если не `test` — `viewModel.startNewAlarm()` (будильник
  перезапустится, если игра не пройдена).
- Каждые две минуты (чётная ненулевая минута, секунды 00, не `test`) —
  повторное включение мелодии через `AlarmMediaPlayer.playAudio` и кнопка
  «Выключить музыку» (`AlarmMediaPlayer.stopAudio`).
- Системная «назад»: в режиме пробы (`test == true`) — возврат к выбору
  игр (`action_moleGameFragment2_to_gameChoiceFragment`); при реальном
  будильнике — ничего не происходит.
- Победа: в аргументы кладутся `time` (String), `score` (Int),
  `game id` = 23; переход к результату — `..._to_gameResultFragment2`
  (проба) или `..._to_gameResultFragment` (будильник) +
  `viewModel.setPositiveResult()`.

## Тесты

`src/test/java/.../MoleLogicTest.kt` (JUnit4, только чистая логика):
параметры всех уровней и приведение вне диапазона; следующая нора в
пределах поля и ≠ предыдущей (1000 итераций на каждый уровень, с seed);
то же после самостоятельного ухода крота; детерминизм при одинаковом
seed; достижимость всех нор; попадание/промах/тап при спрятанном кроте;
отсутствие штрафа за ушедшего крота; победа ровно после N попаданий;
формула бонуса.

```bash
./gradlew :feature:games:mole:test
```

## Запуск в демо

Игра включена в демо-приложение со всеми мини-играми (выбор сложности,
без будильника и Firebase):

```bash
./gradlew :feature:games:demo:installDebug
```
